# parcel-boiler-plate

## Steps

1) Install npm node modules from package.json
```
  npm install
```

2) Run parcel dev server
```
  npm run dev
```



## To run the test 

1) Install Jest Cli
   ```
    npm install jest-cli -g
   ```
   
2) Run spec files
   ```
    jest spec/index.test.js
   ```
